Ready to write code that runs directly on the internet? The DFINITY team has prepared a number of developer resources to help you get started on building decentralized applications native to the Internet Computer.

This category will focus on sharing documentation, tutorials, and how-to information to help you create, deploy, and maintain applications and services that run on the Internet Computer. Within each a sub-category you'll find additional resources from the DFINITY team to help you get started. Feel free to post development focused questions within this category. We also encourage the sharing of tips, hacks and workarounds here within the Developer Resources category.

To get started coding on the Internet Computer, visit the https://sdk.dfinity.org/docs/quickstart/network-quickstart.html quickstart tutorial!
