Ok, for some reason it is not the default. The option is called `gzip`.

```
"canisters": {
    "backend": {
      "type": "rust",
      "package": "backend",
      "candid": "src/backend/backend.did",
      "optimize": "cycles",
      "gzip": true
    }
  }
```
