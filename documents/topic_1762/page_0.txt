dfx canister --help
dfx-canister 
Manages canisters deployed on a network replica

USAGE:
    dfx canister [OPTIONS] <SUBCOMMAND>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
        --network <network>    

SUBCOMMANDS:
    call              Deletes a canister on the Internet Computer network
    create            Creates an empty canister on the Internet Computer and associates the
                      Internet Computer assigned Canister ID to the canister name
    delete            Deletes a canister on the Internet Computer network
