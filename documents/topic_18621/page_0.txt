Agent-JS 0.15.4 is released! This includes subdomain compatibility for new boundary node hosts, and a new `keyType` option for `AuthClient`, allowing you to choose `Ed25519` vs `ECDSA` keys. This is necessary if you are using a `Storage` implementation that doesn't support `CryptoKeys`
